ISTRUZIONI PER IL FILE FIRMA
=============================

In questa cartella va piazzato il file `firma.png` con la firma di Leonardo Massafra.

Procedura (da fare UNA VOLTA SOLA):
1. Salva l'immagine della tua firma come file PNG (o JPG) sul tuo computer
2. Rinomina il file in "firma.png"
3. Copia/trascina "firma.png" in questa cartella (`assets/`)

Caratteristiche ideali:
- Formato: PNG con sfondo trasparente (ottimale) oppure JPG/PNG con sfondo bianco
- Firma di colore nero o blu scuro
- Risoluzione: qualunque, la skill la ridimensiona in automatico

Se il file "firma.png" NON è presente, la skill genera comunque i preventivi
ma SENZA firma grafica (il preventivo sarà solo con "Amministratore – LEONARDO MASSAFRA"
scritto in basso, senza immagine).
